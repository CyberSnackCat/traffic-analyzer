Net Sec Traffic Analyzer

Student-built tool to capture packets, spot suspicious patterns, and generate a simple dashboard.

Get Started Features

Quick Start

  1. Create a Python venv 
    python -m venv .venv
source .venv/bin/activate  # Windows: .venv\Scriptsctivate
pip install -r requirements.txt
  2. Analyze sample data (no admin needed) 
    python app.py analyze --input data/samples/synthetic_log.csv --report

    Open the newest file in reports/.

  3. Live capture (admin/root may be required) 
    python app.py capture --iface YOUR_INTERFACE
python app.py analyze --input data/logs.csv --report

Run only on networks you own or have permission to analyze.

Detections

  • ARP spoofing
  • DNS poisoning
  • Port scans
  • DoS spikes

Reporting

Static HTML report with charts for packets-per-second, top talkers, and DNS queries.

Configurable

Tune thresholds in config.yaml to match your normal traffic baseline.
